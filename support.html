地狱验证码游戏：所有政策与规范

关于游戏

  • 这是一个关于一些特难的验证码游戏，旨在测试人类的极限。
  • 您可以点击下方的“开始游戏”按钮进入挑战。
  • 进入游戏界面后，输入一个任意 Gmail 邮箱（该邮箱不需要真实存在，仅作为存档标识）即可开始。
  • 成功通关后，您可以将成绩加入排行榜。排行榜每天刷新（UTC +8）。

⚠️ 注意事项：挑战过程中可能导致轻微血压升高，若发生意外情况，本游戏概不负责。

您可以做的

  • 认真、诚实地完成游戏，不使用任何形式的作弊手段。
  • 前往 GitHub 提交具有建设意义的 Issue。
  • 申请加入我们的开发者团队。
  • 在 GitHub 上创建 Pull Request，为游戏优化贡献一份力量。

你不应该做的（禁止行为）

  • 通过某些手段强行进入开发者控制台。
  • 使用不正当的辅助脚本或第三方工具通过关卡。
  • 在 GitHub 上提交恶意、无意义或重复的 Issue。
  • 对游戏服务器进行 DDoS 攻击或其他网络破坏。
  • 复制粘贴本游戏内的任何内容。（本页面已启用防复制逻辑）
  • 使用任何影响游戏公平性的手段（如物理辅助、多开等）。

违规处罚

若系统检测到违反上述政策的行为：

  • 您本轮成绩将立即作废，并可能面临设备/IP 的永久封禁。
